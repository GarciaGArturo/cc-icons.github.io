Creative Commons Icons

Yet Another Iconic Font and CSS Toolkit

Download this project as a .zip file Download this project as a tar.gz file

 

 

There are several ways to install and use Creative Commons Icons library, from hosting on your own site like a pro, or by simply adding a few lines to your website source code. Creative Commons Icons plays nice with several CSS and Javascript libs, including Bootstrap.

 

Simple usage

By choosing this method you won't even need to install or download anything, just choose between cdnjs hosted at CloudFlare, or jsDelivr, or RawGit served via MaxCDN or files hosted at Google Drive to add cc-icons to your site with a single line of code.

  1. Copy one of the following HTML codes: <link href="//cdnjs.cloudflare.com/ajax/libs/cc-icons/1.0.1/css/cc-icons.min.css" rel="stylesheet">

    or

    <link href="//cdn.jsdelivr.net/cc-icons/1.0.1/css/cc-icons.min.css" rel="stylesheet">

    or

    <link href="//cdn.rawgit.com/cc-icons/cc-icons/master/css/cc-icons.min.css" rel="stylesheet">

    or

    <link href="//googledrive.com/host/0B8JA5Wj4mv8zRHJTYW51bU05U28/cc-icons.min.css" rel="stylesheet">
  2. Put one of them into your website HTML's<head>section.
  3. All done now check either examples or icons to get to know cc-icons.

Do it like a pro

If you want to host cc-icons on your own website, do the following

  1. Download the zip file above.
  2. Uncompress its content, and upload bothcss/andfonts/folders to your hosting server web root folder.
  3. Find out the actual absolute link to eithercc-icons.cssorcc-icons.min.css. E.g.: considering your website is http://example.com it would result in: <link href="http://example.com/css/cc-icons.min.css" rel="stylesheet">
  4. Once you find the link to one of these files and adapt the code above put it into your website<head>section.
  5. All done! :D

If you are uploading to your hosting server, preserve both folders structure or else you are going to break the library causing malfunction.
